Coaster - EP

By UTS Creations

Download

Page Updated: 10/27/2025

Description

This download contains the new Coaster pack with new audio and skins for the F59PHI and the bi level cars.

Features

  • 3 Brand New Repaints
  • Alerter
  • BiLevel Indicator Lights
  • Clean Cab Radio
  • Door Control
  • Emptying Sander (Feature is optional and can be disabled via the configuration files.)
  • Head-End Power
  • Independent Brake Bail-Off
  • Isolation Switch
  • Pneumatic Control Switch

Requirements

  • San Diego Commuter Rail F59PHI Loco
  • EMD 16N-710G3B Sound Enhancement Pack
