Support Through Ko-fi

This page links to our official Ko-fi page. Any tips or donations are completely optional and only exist to help cover technical costs for running the Train Simulator Archive.

Every donation goes straight toward things like server upgrades, file storage, and ongoing maintenance of this self-hosted archive. I don't make and will never make any personal profit from this. I'm not selling or monetizing anyone's content; I'm simply preserving it so it isn't lost.

If there are no donations, the archive will still stay online. I'm fully committed to keeping it running with my own money if necessary. Ko-fi is just a way for people who want to help share some of the cost.

This project is 100% nonprofit and exists for the community, by the community. Your support is never expected, but always appreciated.
